Ubuntu: Packages
==================================================

The following packages are needed to build the Docbook Exim documentation in
PDF format:

#. ``sudo apt-get update``
#. ``sudo apt-get install git-core docbook docbook-xsl-ns xsltproc fop xfpt
   sdop xmlto libxml2-utils xmlstarlet``
